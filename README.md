### Installation
`sudo apt-get install tmux`

### Configuration

> vi ~/.tmux.conf
> Copy content of .tmux.conf file 

### List of Shortcuts

`prefix` = `Alt + a`

* New horizontal pane - `prefix> %`
* New vertical pane - `prefix> "`
